## 核心叙事逻辑 (Motivation Spine)

```
钠离子电池快充负极存在"倍率-容量-稳定性三者难以兼得"的问题
  → Bi 提供高理论容量，但导电性与体积膨胀是痛点
  → MXene 作为二维导电骨架解决上述两个问题
  → 本文实现了三指标的同步提升
```

## 性能亮点

- 超高倍率下保持高容量
- 超长循环稳定性
- 高首次库仑效率
- 三者兼备（这是核心卖点）

## 表征配置

- 材料表征: SEM, TEM, XRD, XPS
- 电化学: 倍率、循环、CV、EIS 等
- 机理讨论基于非原位表征推理
